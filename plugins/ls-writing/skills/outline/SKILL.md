name: outline
description: Use when content or notes need organizing into a structured, hierarchical outline
---

If the TARGET of this command is not already clear from the context, ask the user: "What would you like to create an outline from?" Substitute TARGET for the user's response below.

Generate a detailed outline from TARGET.

Use bulleted lists with sub-bullets to organize the content.

Create a hierarchical structure with:

- Clear main topics and subtopics
- Logical flow and organization
- Key points and supporting details
- Proper nesting and indentation

Focus on extracting and organizing the core concepts into a coherent framework.
